# Lite DBMS (In Progress)

A lightweight SQL-like database engine built in C++ with file-based storage, record management, and B+ Tree indexing.

## Features (Planned & In Progress)
- CREATE TABLE, INSERT, SELECT operations
- File-based table storage system
- B+ Tree indexing for fast lookups
- SQL-like command parser
- Modular C++ OOP design
